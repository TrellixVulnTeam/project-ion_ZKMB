161  /*
162  * Register a periodic callback for this bus. All callbacks on the
163  * same bus are made from the same thread with lock already taken. In
164  * other words, the callback is not executed on the main thread (or the
165  * thread which registered the callback), but in a separate per-bus
166  * thread.
167  *
168  * After registering the periodic callback, the other functions should not
169  * be used anymore from other contexts. If it really needs to be done, the
170  * lock must be taken.
171  *
172  * Return: A handle for this periodic callback. To cancel the callback
173  * call #unregister_callback() or return false on the callback.
174  */
